# 검강화 게임 추가 아이디어

> 기획서 검토 시 도출된 재미 보강 아이디어. 구현 우선순위는 상단이 높음.
> 채택 시 기획서/구현계획서/태스크에 반영 필요.

---

## 1. "성공했다면" 도발 메시지

파괴 시 팝업에 놓친 이익을 표시:

```
+14 듀랑달이 파괴되었습니다...
💰 성공했다면 판매가: 6,360골드였는데...
```

- "아 한 번만 더..." 심리 유발
- 구현 비용 최소 (텍스트 한 줄), 리텐션 효과 큼

---

## 2. 대장장이 NPC 캐릭터 & 대사

강화할 때마다 대장장이가 한마디:

| 상황 | 대사 예시 |
|------|----------|
| 저단계 강화 | "이 정도는 식은 죽 먹기지!" |
| 고단계 도전 | "+17에 도전? 미쳤..." |
| 성공 | "크으... 명작이야!" |
| 파괴 | "아... 아깝다", "내 실력이 부족했나..." |
| 연속 파괴 | "...오늘은 이만 쉬는 게..." |
| 장인 숙련도 레벨업 | 어조 변화 (초보 → 장인) |

- 단계 구간별 3~5개씩이면 충분
- 감정 이입 + 캐릭터성으로 차별화

---

## 3. 실시간 파괴/성공 피드 (전체 공지)

화면 상단에 다른 유저의 강화 결과가 실시간으로 흘러감:

```
🔥 대장장이#A3F2가 +16 레바테인을 파괴했습니다
⚔️ 대장장이#B7C1이 +13 쿠사나기를 달성했습니다!
```

- "저 사람도 터졌네" → 동질감 + 도파민
- "저 사람은 성공했는데 나도?" → 도전 욕구
- Firebase Realtime DB or FCM 토픽으로 구현 가능
- Phase 3 랭킹과 묶으면 자연스러움

---

## 4. 연속 강화 보너스/벌칙 연출

현재 연속 성공/실패를 통계로만 추적 중. 게임플레이 연출에 반영:

| 조건 | 연출 |
|------|------|
| 연속 성공 3회 | 화면에 불꽃 이펙트 + "불꽃 연속!" |
| 연속 성공 5회 | 대장장이 "오늘 손이 좋은데?!" |
| 연속 파괴 5회 | 대장장이 "...오늘은 이만 쉬는 게..." |
| 연속 파괴 10회 | 화면이 점점 어두워지는 연출 |

- 실제 확률에 영향 없음 — 연출만으로 감정적 경험 차등화
- 이미 데이터 추적 중(Statistics.currentConsecutiveSuccess/Fail)이라 활용만 하면 됨

---

## 5. 마일스톤 보상 (중간 목표)

첫 달성 시 즉각적 보상으로 "다음 마일스톤까지만 해보자" 심리 유도:

| 마일스톤 | 보상 |
|---------|------|
| +5 최초 달성 | 50골드 |
| +10 최초 달성 | 파편 10개 |
| +15 최초 달성 | 보호의 부적 1개 |
| +18 최초 달성 | 축복의 주문서 2개 |
| +20 최초 달성 | 특별 칭호 + 파편 100개 |

- 칭호 시스템과 별개로 즉시 체감 가능한 보상
- 업적 시스템(P3)에 통합 가능

---

## 6. "오늘의 운세" / 일일 보너스

앱 시작 시 오늘의 강화운 연출:

```
⭐⭐⭐☆☆ "오늘의 강화운: 보통"
```

- 방법 A: 순수 연출 (실제 확률 변동 없음, 플라시보)
- 방법 B: 일일 1회 강화 확률 +3%p 보너스 (소소한 실질 혜택)
- 매일 접속 동기 부여 + 첫 강화의 기대감 상승

---

## 7. 검 비주얼 가이드라인

현재 사양서에 검 비주얼 명세 없음. 수집 욕구를 위해 필요:

- 21종 검 각각 고유 일러스트 (최소 단계 구간별 색상/형태 차등)
- 컬렉션에서 "내가 모은 명검들" 감상하는 맛
- 아트 리소스 필요 → 별도 작업 트랙
