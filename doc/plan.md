# 검강화 게임 기획서

## 개요
- **장르:** 캐주얼 강화 시뮬레이션
- **타겟:** 심심풀이 캐주얼 유저
- **수익 모델:** 광고 (리워드 광고 중심)
- **기술 스택:** Flutter
- **플랫폼:** iOS / Android

---

## 핵심 게임 루프

```
나무검 자동 지급 → 강화 → 성공/파괴 → 판매 or 수집 → 나무검 자동 지급 → 반복
```

- 나무검은 항상 무료 자동 지급 (구매 불필요)
- 골드는 강화 비용에만 사용
- 강화 실패 시 즉시 파괴 (단계 하락 없음, 파괴만 존재), 나무검부터 재시작
- 강화 단계가 높을수록 성공률 낮고 비용 비쌈
- 판매가 = 단계별 고정 테이블 (저단계: 투자 대비 70~90% 회수, 고단계: 150%+ 회수 → 고단계 도전에 경제적 동기 부여)

## 강화 단계

> **밸런스 데이터는 외부 파일로 관리:** [`swords.csv`](./swords.csv)
>
> 강화 단계 수, 검 이름, 확률, 비용, 판매가, 수집 여부 등을 CSV에서 수정하면 게임에 즉시 반영.
> 단계 추가/삭제도 행 추가/삭제만으로 가능.

### 데이터 구조 (swords.csv)

| 컬럼 | 설명 |
|------|------|
| 강화 | 강화 단계 (+0 ~ +N) |
| 검 이름 | 해당 단계의 검 이름 |
| 테마 | 검의 출처/테마 (기본, 판타지, 북유럽 신화, 한국 신화, 일본 신화, 서브컬쳐, 뇌절 등) |
| 성공률(%) | 강화 성공 확률 |
| 강화비용 | 이번 단계 강화에 드는 골드 |
| 누적투자 | 여기까지 도달하는 데 든 총 골드 |
| 판매가 | 판매 시 받는 골드 (단계별 고정, 저단계 70%~고단계 200% 회수율) |
| 회수율 | 누적투자 대비 판매가 비율 |
| 파괴시파편 | 파괴 시 획득하는 파편 수 |
| 수집가능 | Y/N — 컬렉션 등록 가능 여부 |

### 검 이름 설계 방향

| 구간 | 테마 | 예시 |
|------|------|------|
| +0 ~ +5 | 기본 판타지 | 나무검, 철검, 황금검 |
| +6 ~ +9 | 판타지/역사 | 미스릴검, 룬소드, 다마스쿠스 블레이드 |
| +10 ~ +14 | 세계 각국 신화/전설 | 엑스칼리버, 그람, 천총운검, 쿠사나기, 듀랑달 |
| +15 ~ +17 | 한국 고전 + 서브컬쳐 | 참마도, 레바테인, 에아 |
| +18 ~ +20 | 뇌절 오리지널 | 초차원 드래곤슬레이어 DX, 울트라 킹왕짱 투명드래곤의 검, ■■■■■■의 검 [데이터 삭제됨] |

- 후반부로 갈수록 이름이 길어지고 과장되는 것 자체가 연출의 일부
- 최종 검은 SCP 스타일 블랙박스 처리로 미스터리 연출

## 기본 시스템

### 화면 구성

```
[타이틀] → [대장간 (메인)]
                ↕
            [공방] ← 장인 숙련도 + 파편 교환 + 컬렉션
                ↕
            [업적/칭호]
```

| 화면 | 역할 | 주요 요소 |
|------|------|----------|
| 타이틀 | 앱 진입 | 게임 시작, 설정 |
| 대장간 (메인) | 핵심 게임 플레이 | 현재 검, 강화 버튼, 골드, 확률, 판매/수집 버튼 |
| 공방 | 성과 확인 + 자원 관리 | 장인 숙련도/경험치, 파편 교환소, 검 컬렉션 도감 |
| 업적/칭호 | 목표 리스트 | 달성/미달성 업적, 획득 칭호, 칭호 장착 |

- 대장간 화면 하단 네비게이션으로 공방/업적 접근
- 공방 = "마을", 대장간 = "전투" 느낌
- 컬렉션은 공방 내 탭/스크롤로 배치 (대장장이가 만든 명검 진열 세계관)

### 앱 흐름
- 타이틀 화면 → 대장간 화면 (나무검부터 시작)
- 골드, 컬렉션, 업적, 통계, 장인 숙련도, 파편은 이전 플레이에서 이어짐
- 현재 검 상태는 저장하지 않음 (매번 나무검부터)

### 초기 설정
- 초기 자금: 200골드
- 나무검은 무료 자동 지급 (파괴 시에도 즉시 재지급)

### 강화 선택지
- +9 이하: [강화] [판매]
- +10 이상: [강화] [판매] [수집]

### 판매
- 판매가 = 단계별 고정 테이블 (swords.csv에서 관리)
  - 저단계(+1~+7): 투자 대비 70~90% 회수 → 손절 수단
  - 중단계(+8~+12): 투자 대비 100~130% 회수 → 본전~소폭 이익
  - 고단계(+13~+20): 투자 대비 150~200% 회수 → 고위험 고수익
- 판매 후 골드 획득, 나무검부터 재시작

### 수집
- +10 이상 검만 수집 가능
- 수집 시 골드 없음 (판매와 택1)
- 수집 후 컬렉션에 등록, 나무검부터 재시작
- 수집 가능 검: 11종 (엑스칼리버 ~ 광선검)

## 공방 화면 상세

공방은 3개 영역으로 구성 (탭 또는 스크롤):

### 장인 숙련도
- 숙련도 레벨 + 경험치 바 (다음 레벨까지 N회)
- 다음 레벨 보상 미리보기
- 공방 외형 (레벨에 따라 변화)

### 파편 교환소
- 보유 파편 수 표시
- 교환 가능 아이템 목록 (보호의 부적 / 축복의 주문서 / 골드 주머니)
- 보유 아이템 수량 표시

### 컬렉션 (검 도감)
- 수집한 검은 공방 벽면에 진열된 형태로 감상
- 미수집 검은 "???" 실루엣으로 표시
- 컬렉션 완성도 표시 (예: 3/11, 27%)
- 같은 검 중복 수집 시 별 표시 (★★★)
- 전체 도감 완성 시 특별 칭호

## 광고 시스템

| 타이밍 | 형태 | 보상 | 제한 |
|--------|------|------|------|
| 파괴 시 | 리워드 광고 | 검 복구 (보호권) | 하루 2회 |
| 골드 소진 시 | 리워드 광고 | 200골드 | 무제한 |
| 강화 전 | 리워드 광고 (선택) | 확률 +5%p 부스터 | 무제한 |

- **광고 보호권**(검 복구)은 일일 2회로 제한 → 희소성 부여, 고단계에서 전략적 사용 유도
  - 파괴 직후 사후 적용: 파괴 판정 → 보호권 사용 여부 팝업 → 사용 시 파괴 취소, 현재 단계 유지
- 골드 광고는 무제한 → 골드 데드락 방지 (단, 200골드 고정이므로 고단계에선 여러 번 시청 필요)
- **광고 부스터**는 절대값 +5%p (예: 기존 10% → 15%), 다음 1회 강화에만 적용, 강화 전 사전 적용
- 핵심: 유저가 자발적으로 광고를 보고 싶어하는 구조

### 용어 정리 (광고 아이템 vs 파편 아이템)

| 용어 | 획득 경로 | 효과 | 비고 |
|------|----------|------|------|
| 광고 보호권 | 파괴 시 리워드 광고 | 파괴 취소, 현재 단계 유지 (사후 적용) | 일일 2회 제한 |
| 광고 부스터 | 강화 전 리워드 광고 | 확률 +5%p (사전 적용, 1회) | 무제한 |
| 보호의 부적 | 파편 30개 교환 | 다음 강화 실패 시 파괴 대신 단계 유지 (사전 적용, 1회) | 소모품 |
| 축복의 주문서 | 파편 15개 교환 | 다음 강화 확률 +5%p (사전 적용, 1회) | 소모품 |
| 골드 주머니 | 파편 10개 교환 | 즉시 500골드 획득 | 소모품, ExchangeCommand로 처리 |

- 광고 보호권과 보호의 부적은 **효과는 같지만 획득 경로와 적용 시점이 다름**
  - 광고 보호권: 파괴 후 팝업에서 사후 적용
  - 보호의 부적: 강화 전 사전 적용 (부적 소모 후 강화 → 실패해도 파괴 안 됨)
- 광고 부스터와 축복의 주문서는 **효과 동일(+5%p), 중복 적용 가능 (최대 +10%p)**

## 추가 시스템

### 장인 숙련도

세션은 리셋되지만 장인 숙련도는 영구 성장. 강화를 할수록 대장장이로서 성장하는 메타 진행 시스템.

- **경험치 기준**: 강화 시도 횟수 (성공/실패 무관) → "실패해도 성장한다"
- 장인 숙련도와 보상은 외부 데이터로 관리 가능

| 장인 숙련도 | 필요 경험치 (누적 강화 횟수) | 보상 |
|------------|--------------------------|------|
| Lv.1 초보 장인 | 시작 | - |
| Lv.2 | 50회 | 강화 비용 5% 할인 |
| Lv.3 | 150회 | +1~+3 강화 연출 스킵 옵션 |
| Lv.4 | 400회 | 강화 비용 10% 할인 |
| Lv.5 | 800회 | 공방 외형 변경 (용광로 추가) |
| Lv.6 | 1,500회 | 강화 비용 15% 할인 |
| Lv.7 | 3,000회 | 파편 획득량 +1 |
| Lv.8 | 5,000회 | 강화 비용 20% 할인 |
| Lv.9 | 8,000회 | 공방 외형 변경 (전설의 공방) |
| Lv.10 | 15,000회 | 강화 비용 25% 할인 + 전용 칭호 "마스터 스미스" |

### 파편 시스템

강화 실패(파괴) 시 "검의 파편"을 획득. 파괴가 완전한 손실이 아닌 장기적 자원이 되는 구조.

- 파편은 영구 누적 (세션 리셋과 무관)
- 파괴된 검의 단계에 비례하여 파편 획득 (swords.csv의 "파괴시파편" 컬럼)

| 파괴 단계 | 기본 획득 파편 |
|----------|--------------|
| +1 ~ +5 | 1개 |
| +6 ~ +9 | 3개 |
| +10 ~ +14 | 8개 |
| +15 ~ +17 | 20개 |
| +18 ~ +20 | 50개 |

※ 장인 숙련도 Lv.7 이상 시 파편 획득량 +1

**파편 사용처**:

| 아이템 | 파편 비용 | 효과 |
|--------|----------|------|
| 보호의 부적 | 30개 | 다음 강화 실패 시 파괴 대신 현재 단계 유지 (1회) |
| 축복의 주문서 | 15개 | 다음 강화 확률 +5%p |
| 골드 주머니 | 10개 | 500골드 획득 |

### 확률 표시
- +1~+14: 성공률 숫자로 표시
- +15 이상: "???"로 숨기기 (스릴 강화)

## 데이터 저장 및 동기화

### 저장 구조

```
[유저 폰 - 로컬]              [Firebase - 서버]
├── 골드              ←동기화→  ├── 골드
├── 컬렉션            ←동기화→  ├── 컬렉션
├── 업적/칭호          ←동기화→  ├── 업적/칭호
├── 통계              ←동기화→  ├── 통계
├── 장인 숙련도/경험치  ←동기화→  ├── 장인 숙련도/경험치
├── 파편 보유량         ←동기화→  ├── 파편 보유량
└── 광고 시청 횟수              └── 랭킹
```

- 현재 검 상태는 저장하지 않음 (앱 시작 시 항상 나무검부터 시작)
- 통계: 역대 최고 강화, 총 파괴 횟수, 총 강화 시도, 연속 성공/파괴 최대 기록 등 (랭킹 산출용)
- 장인 숙련도/경험치: 영구 누적
- 파편: 영구 누적

- 로컬: SharedPreferences 또는 Hive (빠른 읽기/쓰기, 오프라인 가능)
- 서버: Firebase (Firestore + Auth 익명 로그인)

### 동기화 타이밍

**서버 → 로컬 (다운로드)**
- 앱 시작 시 (다른 기기 데이터 반영)

**로컬 → 서버 (업로드)**
- 판매 시
- 수집 시
- 파괴 시
- 앱 백그라운드 전환 시 (홈 버튼)

강화 자체는 연타로 빠르게 누르는 게임이므로, 강화할 때마다 서버 호출하면 딜레이가 생겨 답답함. 판매/수집/파괴 등 결과가 확정되는 시점에만 동기화.

### Firebase 비용

| 항목 | 무료 한도 | 예상 사용량 |
|------|----------|-----------|
| 저장 용량 | 1GB | 유저 1만명이어도 수십MB |
| 쓰기 | 일 2만회 | 유저당 하루 수십회 |
| 읽기 | 일 5만회 | 유저당 하루 수회 |

## 개발 우선순위

### Phase 1 (MVP)
- 강화/파괴 핵심 루프
- 검 판매
- 20단계 검 진화
- 골드 시스템
- 기본 UI/연출

### Phase 2
- 컬렉션 시스템
- 파편 시스템
- 장인 숙련도 시스템
- 로컬 기록 (최고 강화, 파괴 횟수 등)
- 광고 연동 (AdMob)

### Phase 3
- 랭킹 시스템 (Firebase)
- 칭호 시스템
- 업적
- 푸시 알림
